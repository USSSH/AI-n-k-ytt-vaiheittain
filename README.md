## Projektin kuvaus
Tämän projektin tavoitteena on toteuttaa (Booking Logic) ydin tekoälyn avulla vaiheittain ja harkitusti.
Painopiste ei ole täydellisen sovelluksen rakentamisessa, vaan pikemminkin varauslogiikan ymmärtämisessä, häiriöiden estämisessä ja tekoälyn käytön dokumentoinnissa kehityksen aikana.

## Oppimistavoite
Projektin tavoitteet ovat: - Oppia käyttämään tekoälyä tietoisesti
- Ylläpitää koodin täydellistä ymmärtämistä
- Kyky selittää ja perustella ohjelmistoratkaisuja
- Yhdistää BDD varsinaiseen toteutukseen
